# AutoTable

AutoTable:

## 何时使用

当你有一些表格数据展示是基于内置的一些 CRUD 的操作，可以以一种通用的 RESTful 风格的数据操作 API 进行与服务端进行交互，或者需要多种单元格样式时，AutoTable 是不错的选择。
